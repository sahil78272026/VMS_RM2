🔑 Main Features

1. Logging Visits

  • Tracks when visitors arrive and depart.
  • Captures visitor details like name, reason for visit, and flat number.

2. Resident Consent for New Visitors

  • For first-time visitors, the system sends a consent request to the resident.
  • Access is granted only once the resident approves.

3. Recognising Regular Guests

  • Frequent visitors (domestic workers, delivery staff) can be auto-approved.
  • Minimises unnecessary alerts for residents.

4. Alerts for Unsuitable Visitors

  • If a visitor is flagged as unsuitable, the system notifies residents and security.
  • Enhances safety and accountability in the community.
